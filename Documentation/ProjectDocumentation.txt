About the project Searchify:
	Followed clean architecture
	Mediator pattern (used MediatR library)
	JWT Authentication
	MS SQL Server
	Xunit testing framework

Project Structure:
	Core
	    SearchDemo.Domain 
            SearchDemo.Application
	Infrastructure
  	    SearchDemo.Infrastructure
	Presentation
	    SearchDemo.API
	Test 
	    SearchDemo.Test
 

Pre-requisite to run the application:
	a) Visual studio 2022 (.NET Core 8)
